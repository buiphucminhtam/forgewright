Quick Start

Get Running in Minutes

Three ways to integrate Forgewright into your workflow.

Terminal
$ curl -sO https://raw.githubusercontent.com/buiphucminhtam/forgewright/main/setup.sh
$ chmod +x setup.sh && ./setup.sh install

# Then add to your project:
$ ./forgewright/setup.sh init

Setup Options

Git Submodule

Add as a submodule for team-wide consistency. Perfect for shared projects.

Standalone

Use Forgewright as the main project. Best for solo developers or dedicated setups.

MCP Server

Connect via MCP for IDE integration with Cursor, Claude Code, and more.
